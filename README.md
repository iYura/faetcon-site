# FAETCon — сайт конференції

Сайт конференції **FAETCon** — бейдж-хакінг, електроніка та авіоніка.
Організатор — Факультет аеронавігації, електроніки та телекомунікацій (ФАЕТ),
Київський авіаційний інститут. За мотивами Hackaday Supercon.

**28 жовтня 2026 · КАІ, Київ**

## Структура

- `docs/` — готовий статичний сайт (його показує GitHub Pages)
- `src/content/uk/`, `src/content/en/` — контент сторінок (редагувати тут)
- `src/templates/base.html` — спільний каркас сторінки (шапка, навігація, підвал)
- `src/build.py` — генератор (без залежностей)

## Як редагувати

1. Змінити текст у `src/content/uk/...` та/або `src/content/en/...`
2. Перезібрати: `cd src && python3 build.py`
3. Закомітити й запушити — GitHub Pages оновиться сам
